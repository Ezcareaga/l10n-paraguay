Foundational module for the Paraguayan localization. Provides the canonical
DNIT/SIFEN catalogs (departments, districts, cities, regimes, taxpayer types,
identification types) and tax-ID validations (CI, RUC) using the official
Paraguayan módulo 11 algorithm.

Usage
=====

After installation, navigate to *Contacts > Paraguay* to inspect the loaded
catalogs.  When creating partners whose country is Paraguay, select the proper
identification type (RUC, Cédula paraguaya, Pasaporte, etc.) and the module
will validate the format automatically.
