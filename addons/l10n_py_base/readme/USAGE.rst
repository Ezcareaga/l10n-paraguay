After installation, navigate to ``Contacts > Paraguay`` to inspect the loaded
catalogs (regimes, taxpayer types, districts).

When creating a partner whose country is Paraguay:

#. Select an identification type from the list provided
   (``RUC``, ``Cédula paraguaya``, ``Pasaporte``, etc.).
#. Enter the document number in the ``VAT`` field.
#. The module validates the format automatically when saving:

   * RUC: must follow the format ``<digits>-<dv>`` and the verification
     digit must match the official módulo 11 algorithm.
   * Cédula paraguaya: must be numeric (1 to 9 digits).
