## Cuando se generen

Schema canónico propuesto:

```csv
codigo,nombre,nombre_display,codigo_padre,activo
1,CAPITAL,Asunción,,1
2,CONCEPCION,Concepción,,1
...
```

Columnas:

| Campo | Tipo | Notas |
|---|---|---|
| `codigo` | int | Código SIFEN — el que va en el XML del DE |
| `nombre` | str | UPPERCASE ASCII-safe (para matching exacto) |
| `nombre_display` | str | Con tildes, capitalización title-case (para UI) |
| `codigo_padre` | int / null | Para hijos (distrito → depto, ciudad → distrito) |
| `activo` | 0/1 | `1` vigente, `0` dado de baja por DNIT |
