## Schemas observados

### DNIT XLSX

Sheet: `División Político-Administrativa`

```
[blank] | Código Depto | Departamento | Código Distrito | Distritos | Código Ciudad | Ciudades/Localidades | Código Barrio | Barrios
```

Desnormalizado: cada fila tiene los 4 niveles. La col 1 está vacía
(probablemente artefacto del template Excel del DNIT). Datos repetidos verticalmente
por cada barrio.

### INE 2022 (XLSX)

Departamentos:

```
COD_DEP (str 2 dígitos) | DEPARTAMENTO
```

Distritos:

```
Idx | CÓDIGO (4 dígitos = depto+distrito) | COD_DEP | DEPARTAMENTO | COD_DIS | DISTRITO
```

Barrios:

```
COD_DEP | COD_DIS | COD_AREA | COD_BAR_LOC | BARRIO_LOCALIDAD | DEPARTAMENTO | DISTRITO
```

### INE 2012 (CSV)

Departamentos (`;`-separated, encoding latin-1):

```
codigo_dpto;descripcion_dpto
```

Distritos:

```
Codigo concatenado;Codigo de Departamento;Descripcion de Departamento;Codigo de Distrito;Descripcin de Distrito
```

Barrios:

```
Codigo concatenado;Codigo de Departamento;Descripcion de Departamento;Codigo de Distrito;Descripcion de Distrito;Area;Codigo de Barrio/Localidad;Descripcion de Barrio/Localidad
```

### CONACYT

Sheet: `Hoja1`. Title row + header en row 3:

```
DEPARTAMENTO | DISTRITO | CODIGO DEPARTAMENTO | CODIGO-DISTRITO | COD_2
```

`COD_2` = `CODIGO DEPARTAMENTO + CODIGO-DISTRITO` (concatenado 4 dígitos).

## Discrepancias críticas

Resumen ya documentado en `../README.md`. TL;DR:

- DNIT vs INE: **numeración distinta** de departamentos. DNIT: 1-18. INE: 00-17.
- DNIT tiene **26-34 distritos más** que INE → DNIT incluye subdivisiones
  internas o municipios creados después de 2022.
- Encoding INE 2012 CSVs: latin-1 (caracteres especiales mal codificados al leer
  como UTF-8). Re-encodear al normalizar.
